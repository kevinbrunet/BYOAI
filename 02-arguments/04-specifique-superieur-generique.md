# Argument 4 — Le spécifique redevient supérieur au générique par défaut

## Le raisonnement

Le générique n'a jamais été un idéal, c'est un compromis coût/bénéfice qu'on acceptait faute de mieux. Le sur-mesure a toujours été préférable dans l'absolu.

## Conséquence

Si le coût qui justifiait le compromis disparaît (voir [argument 1](./01-effondrement-economie-echelle-code.md)), le compromis n'a plus de raison d'être imposé uniformément.
